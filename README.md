
# WeatherFi

## Summary
### A Progressive Web App Built using React Js.  The user can see the current weather, along with a 3 day forecast.  Users can search by city or zip code.

## Technologies 
* React.js
* Axios
* WeatherAPI
* ServiceWorker
* Moment Js
